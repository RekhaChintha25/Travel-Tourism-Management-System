# Travel & Tourism Management System

## Project Overview
The Travel & Tourism Management System is a web-based application developed using Java Full Stack technologies. It allows users to register, log in, view travel packages, book packages, make payments, and manage their profiles.

## Technologies Used
- Java
- JSP
- Servlets
- JDBC
- MySQL
- HTML
- CSS
- JavaScript
- Apache Tomcat

## Features
- User Registration & Login
- View Travel Packages
- Book Packages
- Secure Payment
- User Profile Management
- Booking History
- Change Password
- Admin Management

## Project Structure
- Frontend: JSP, HTML, CSS, JavaScript
- Backend: Java, Servlets
- Database: MySQL
- Connectivity: JDBC
